ISIP
====

Scope
-----

ISIP (Infrastructure and Software Integration Project) is in charge of
developing and maintaining the EU-IM-TF simulation infrastructure.

Project Timeline
----------------

The ISIP Timeline can be found `here <../imports/isip_timeline.pdf>`__.
