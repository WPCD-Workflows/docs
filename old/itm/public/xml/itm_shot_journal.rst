Shot Journal
============

The EU-IM has generated a series of publically available data base
entries, so called shots, which can be accessed on the EU-IM Gateway via
the UAL (Universal Access Layer). Shots in private data bases are listed
here as well, however, they are not validated. They may be used for
testing purposes and code development though.
