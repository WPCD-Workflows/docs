Infrastructure
==============

The part of the EU-IM documentation addresses infrastructure related
issues like the EU-IM Gateway, EU-IM database access, the Universal Access
Layer (UAL), and useful tools for handling CPOs.

Mailing Lists
-------------

A number of EU-IM discussion mailing lists have been created:

-  itm-c_and_cpp
   : To share C and C++ related topics
-  itm-data-struct
   : Discussion about changes to EU-IM data structures
-  itm-fortran
   : Use of FORTRAN within the EU-IM
-  itm-hpio
   : Discussion about High Performance I/O
-  itm-matlab
   : Use of MATLAB within the EU-IM
-  itm-plot
   : Discussion of visualization in the EU-IM
-  itm-python
   : Use of Python within the EU-IM
-  itm-tools
   : Discussion of EU-IM tool needs
-  itm-workflow
   : Discussion of issues related to workflows

These lists can be joined by clicking
`here <https://gforge6.eufus.eu/project/itm/mailman/>`__ and then
clicking on "Subscribe/Unsubscribe/Preferences".

The ETS also has two mailing lists:

-  ets-announce
   : ETS announcements list
-  ets-discuss
   : ETS discussion list

These lists can be joined by clicking
`here <https://gforge6.eufus.eu/project/ets/mailman/>`__ and then
clicking on "Subscribe/Unsubscribe/Preferences".

AMNS also has a mailing list:

-  amns-discuss
   : AMNS Discussion List

This list can be joined by clicking
`here <https://gforge6.eufus.eu/project/amns/mailman/>`__ and then
clicking on "Subscribe/Unsubscribe/Preferences".
