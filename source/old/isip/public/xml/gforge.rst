GForge Documentation
--------------------

The GForge Group L.L.C. has published extensive documentation material
in the form of the `GForge AS User
Manual <../imports/GFAS_User_Manual_5.4.pdf>`__ and the `GForge AS
Project Administrator
Manual <../imports/GFAS_Project_Admin_Manual_5.4.pdf>`__. Please refer
to these manual if in doubt on how to use a feature of the EU-IM GForge
system.
