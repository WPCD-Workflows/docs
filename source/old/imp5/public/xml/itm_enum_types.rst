Enumerated datatypes
====================

UNDER CONSTRUCTION

In the EU-IM datastructures there are a number of type-identifiers to
specify, e.g. if the heating source originate from an IC wave or beam
injection, or if a transport coefficient is neoclassical or gyro-Bohm.
In cases where there is a finite number of alternative types, then these
can be numbered. Below is a list of the specified data types used in
